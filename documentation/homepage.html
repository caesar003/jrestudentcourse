This class would treat user diferently whether they are an admin or a regular user.

Shedule Admin

An administrator have access to change schedules, modify teachers, or even add new teachers as well as delete them.

Schedule User

A regular user would only be able to read the schedule for the current date, or for other days (if the schedule for requested day has been created by administrator).
