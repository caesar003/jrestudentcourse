Index

  function index(){
    $this->load->view('elegant_login.php');
    if($this->session->userdata('logged_in') == TRUE){
      redirect('student');
    }
  }

Those line above load the form on elegant_login.php and check if user has already logged in, if user has, it will redirect the page to the homepage.
